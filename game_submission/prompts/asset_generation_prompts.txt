===========================================
ECOPOLIS: ASSET GENERATION PROMPTS
Used for visual and audio assets
===========================================

PROMPT 1: Color Scheme
-----------------------
"Design a color palette for an environmental board game:
- Primary color: eco-friendly green
- Secondary colors for different zones:
  * Energy (orange/yellow)
  * Urban (blue)
  * Natural (green)
  * Policy (purple)
- Accent colors for alerts and highlights
- Dark mode friendly background
- High contrast for accessibility
- Professional but playful tone"

PROMPT 2: Player Tokens
------------------------
"Choose 6 emoji or icon sets for player tokens that represent:
- Environmental/nature themes
- Easily distinguishable from each other
- Work well at different sizes
- Examples: plants, animals, natural elements
- Should be: 🌿 🌱 🌳 🍃 🌾 🌻"

PROMPT 4: Card Design
----------------------
"Design card layout template:
- Card background with gradient
- Border indicating card type (green/orange/purple)
- Icon at top
- Title text
- Description text
- Effect summary at bottom
- Decorative elements (corners, patterns)

Use CSS gradients and borders, no images needed"

PROMPT 5: UI Components
------------------------
"Design modern UI components:
- Buttons (primary, secondary, action)
- Input fields and select dropdowns
- Modal dialogs with backdrop
- Player stat panels
- Activity log messages
- Progress indicators
- Tooltips

Style: Glassmorphism with backdrop blur, rounded corners, subtle shadows"

PROMPT 6: Game Board Layout
----------------------------
"Design a square board game layout:
- 40 spaces around perimeter (10 per side)
- Each space 70-80px square
- Central area for dice display
- Space for card decks
- Player token placement
- Clear ownership indicators
- Zone color coding
- Corner spaces more prominent"

PROMPT 7: Typography
--------------------
"Choose fonts and text styling:
- Headings: Bold, modern sans-serif
- Body: Clean, readable sans-serif
- Numbers: Tabular figures for alignment
- System fonts: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Font sizes: 
  * Title: 4rem
  * Heading: 1.8rem
  * Body: 1rem
  * Small: 0.85rem"

PROMPT 8: Animations
--------------------
"Design animation effects:
- Dice roll: 360° rotation with bounce
- Card reveal: Slide down + fade in
- Player movement: Smooth arc with hop
- Modal open: Fade background + scale content
- Button hover: Lift up + glow
- Number changes: Count up animation
- New log entry: Slide from right

Timing: 0.3-0.5s, easing: ease or ease-out"

PROMPT 12: Responsive Breakpoints
----------------------------------
"Define responsive design breakpoints:
- Desktop large: 1400px+
  * 3 columns, full board (700px)
- Desktop: 1024-1399px
  * 3 columns, medium board (600px)
- Tablet: 768-1023px
  * Stacked layout, small board (500px)
- Mobile: <768px
  * Single column, compact board (400px)
  * Touch-optimized controls"

PROMPT 18: Accessibility
-------------------------
"Ensure visual accessibility:
- Minimum contrast ratio 4.5:1
- Color not sole indicator (use icons too)
- Text size minimum 14px
- Touch targets minimum 44px
- Keyboard navigation visible
- Screen reader friendly labels
- Reduced motion option
- High contrast mode support"
